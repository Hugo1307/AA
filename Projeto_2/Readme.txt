Relatório:

 - O relatório em PDF encontra-se dentro do diretório /report.

Resultados:

 - Os resultados obtidos para as execuções dos algorítmos podem encontrar-se dentro do diretório /results.

Dados de teste:

 - Os dados de teste (grafos gerados para testar os algoritmos) encontram-se dentro do diretório /graph/data. Estão em formato pickle para que possam ser carregados
   e guardados facilmente usando a biblioteca Pickle do python.
